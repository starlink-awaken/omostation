# omostation — Personal AI Operating System

> 4+1+3+I 架构 · 25 项目 · 9.4/10

## Architecture

| Layer | Name | Description |
|-------|------|-------------|
| P0 | Product | Web/CLI/Browser interfaces |
| I0 | Integration | MCP routing, operations, monitoring |
| L4 | Self | Identity, vision, cognitive frameworks |
| L3 | Collaboration | Task management, phase locking, tracing |
| L2 | Capability | Agents, inference, tools, memory, ETL |
| L1 | Contract | Schema, configuration, protocols |
| X1 | Governance | Constraints, secrets, security |
| X2 | Anti-entropy | Freshness, backup, review |
| X3 | Value | Consensus, traceability |

## Quick Start

```bash
make setup   # Install dependencies
make build   # Build all projects
make test    # Run all tests
```
