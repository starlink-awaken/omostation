## 一、Chezmoi 配置管理

### 架构

```
MBP M5 Max                     Mac mini
──────────                     ────────
源端（写配置）                   接收端（只拉不改）
chezmoi add ~/.zshrc            chezmoi update
git push                        chezmoi apply
                                └→ 触发 run_onchange_ 钩子
         │                              │
         └────────── GitHub ────────────┘
                 私有仓库
            starlink-awaken/dotfiles
```

### 仓库结构

```
~/.local/share/chezmoi/
├── dot_zshrc.tmpl               ← Zsh 配置（主机名分支）
├── dot_gitconfig.tmpl            ← Git 配置
├── dot_tmux.conf                 ← Tmux 配置
├── dot_ssh/
│   └── config.tmpl               ← SSH 配置（1Password 注入）
├── dot_config/
│   ├── nvim/                     ← Neovim 配置
│   ├── aerospace/                ← 窗口管理器
│   ├── karabiner/                ← 键盘映射
│   └── hermes/
│       └── config.yaml.tmpl      ← Hermes 配置
├── private_dot_env               ← 敏感变量（age 加密）
├── run_once_before_brew.sh.tmpl  ← Homebrew 初始化
├── run_onchange_brewfile.sh.tmpl ← Brewfile 变更触发
└── run_onchange_macos_defaults   ← macOS 偏好设置
```

### 模板示例

```zsh
# .zshrc.tmpl — 根据主机名自动适配
{{ if eq .chezmoi.hostname "Mac-mini-2" }}
  # Mac mini：本机起服务
  export OLLAMA_HOST=127.0.0.1:11434
  export ONE_API=http://127.0.0.1:3000/v1
{{ else if eq .chezmoi.hostname "MBP-M5" }}
  # MBP：远程调 Mac mini
  export OLLAMA_HOST_MM=100.x.x.1:11434   # Mac mini 轻型
  export OLLAMA_HOST_LOCAL=127.0.0.1:11434 # 本机重型
  export ONE_API=http://100.x.x.1:3000/v1
{{ end }}
```

### 敏感信息

```
不存入 Git：SSH 私钥、API Key、1Password 会话
处理方式：
  - 1Password CLI 动态注入（`op item get`）
  - age 加密（chezmoi 自动加解密）
  - .env 文件模板渲染
```

## 二、Hermes Agent

| 设备 | 装 Hermes？ | 理由 |
|------|------------|------|
| MBP M5 Max | ✅ **唯一一台** | 主控终端 |
| Mac mini | ❌ | SSH 远程管即可 |
| Y7000P | ❌ | Windows 不支持 |
| 单位台式机 | ❌ | 纯网关 |

### Hermes 配置关键点

```yaml
# config.yaml 中 provider 指向 New API
providers:
  new-api:
    base_url: http://100.x.x.1:3000/v1   # Mac mini
    api_key: sk-hermes-token              # New API 生成的专用 Key
    default_model: deepseek-v4-flash

  macmini-ollama:
    base_url: http://100.x.x.1:11434/v1   # Mac mini 轻量推理
    api_key: ollama

  mbp-ollama:
    base_url: http://127.0.0.1:11434/v1   # MBP 本地重型推理
    api_key: ollama
```

### 模型优先级

```
Hermes 调用逻辑（自动选择）：

1. 简单问题 / 代码补全         → New API → DeepSeek V4 Flash
2. 复杂分析 / 架构设计         → New API → DeepSeek V4 Pro / GLM
3. 长上下文文档                → New API → Kimi K2
4. 高并发批量                 → New API → MiniMax M2.5
5. 离线 / 隐私 / 无网环境      → 本地 Ollama（按优先级选）
   a. MBP 本地有重型模型        → MBP Ollama (127.0.0.1)
   b. 否则 → Mac mini 轻量推理   → Mac mini Ollama (100.x.x.1)
```
