## 二、三态切换（MBP）

| 状态 | Llama 调用 | One-API 调用 | 存储 |
|------|-----------|-------------|------|
| **🏠 在家** | Mac mini:11434 (轻) + MBP 本地 (重) | Mac mini:3000 | 内置 |
| **🚗 出差** | MBP 本地 (三星 SSD 上的模型) | 不可达，全离线 | **三星 SSD** |
| **🏢 单位** | Mac mini:11434 (远程) | Mac mini:3000 | 内置 |

### 切换脚本

```bash
# ~/bin/switch-mode.sh
#!/bin/bash
case "$1" in
  home)
    export OLLAMA_HOST=127.0.0.1:11434
    export OLLAMA_MM=100.x.x.1:11434
    echo "🏠 Home: 本地重型 + Mac mini 轻型 + New API"
    ;;
  travel)
    export OLLAMA_MODELS=/Volumes/DevDrive/Ollama-Models/
    export OLLAMA_HOST=127.0.0.1:11434
    echo "🚗 Travel: 三星 SSD 离线推理"
    ;;
  office)
    export OLLAMA_HOST=100.x.x.1:11434
    export ONE_API=http://100.x.x.1:3000/v1
    echo "🏢 Office: 远程 Mac mini + New API"
    ;;
esac
```

## 三、日常操作清单

### 每天

```
□ MBP 接电 + 插三星 SSD（如需出差）
□ Tailscale 自动连（后台）
□ 开始开发
  └ IDE 自动通过 New API 调云端模型
  └ 需要重型离线推理 → 调 MBP 本地 Ollama
```

### 每周

```bash
□ 检查 Tailscale 连接
  tailscale status               # 确认 direct 非 relay

□ 同步配置
  chezmoi update                  # Mac mini 拉取 MBP 配置变更

□ 检查备份
  tmutil destinationinfo          # Time Machine 最近备份

□ Google One 空间
```

### 每月

```bash
□ Docker 清理
  docker system prune -a --volumes

□ Ollama 更新
  ollama pull qwen2.5:32b        # 拉取新版

□ New API 更新
  docker pull calciumion/new-api:latest
  docker restart new-api

□ Chezmoi 整理
  cd ~/.local/share/chezmoi
  chezmoi managed                 # 检查追踪文件
```

## 四、故障排查

| 问题 | 排查命令 |
|------|---------|
| One-API 不通 | `curl http://100.x.x.1:3000/api/status` |
| Ollama 不通 | `curl http://100.x.x.1:11434/api/tags` |
| Tailscale 不通 | `tailscale ping 100.x.x.1` |
| SMB 备份不通 | `open smb://100.x.x.3/Mac_TimeMachine` |
| 代理失效 | `curl -x http://127.0.0.1:7890 https://www.google.com` |
| 模型路由异常 | 登录 New API 面板 → 日志 → 查看调用记录 |
