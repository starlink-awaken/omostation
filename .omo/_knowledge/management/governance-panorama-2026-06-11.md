### R0-R5 健康度评分 (§17.4 公式 R40 修订)

```
debt_density = (drift_count - locked_drift) / total_records
   ↓
if density <= 0.01:    R0 优秀
elif density <= 0.05:  R1 健康
elif density <= 0.10:  R2 警告
elif density <= 0.30:  R3 严重
elif density <= 0.50:  R4 危急
else:                   R5 失控
```
