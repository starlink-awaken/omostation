## 1. 任务完成概览
- **[M1] Ghost Code Exorcism**: 清理了 `agent-runtime` 幽灵代码，并成功修复相关降级/连通性报错。
- **[M2] Plugin Manifest Schema**: 在 `plugin.yaml` 中新增 `metaos_admission` 字段，并在 `agora/mcp_registry/lifecycle.py` 中注入兜底属性，确保受限服务（如 `github-source`）能平滑通过 MetaOS 的校验。

  - 修复 `omo_audit.py` 在嵌套 `asyncio` 环境下的 `RuntimeError` 以及未等待协程的警告。
