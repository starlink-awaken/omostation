## 方法

1. 增强 `bin/gac/kos-seed-import.py`：
   - `--prefer-new`：limit 只消耗「DB 中尚无」路径
   - `--root` / `--roots-only`：指定 vault 精选面
   - 默认 vault 集合增加 `@工作文档`、`@驾驶舱`
2. 执行：

```bash
python3 bin/gac/kos-seed-import.py \
  --db kos/kos-index.sqlite \
  --prefer-new --limit 800 \
  --roots-only \
  --root ~/Documents/@工作文档 \
  --root ~/Documents/@驾驶舱
# → indexed=800 total_documents=3226
# 二次 prefer-new 小批 → 3231
```

3. 计数：

```bash
python3 -c "import sqlite3; print(sqlite3.connect('kos/kos-index.sqlite').execute('select count(*) from documents').fetchone()[0])"
# 3231
```
