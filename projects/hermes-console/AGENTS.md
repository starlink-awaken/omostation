# AGENTS.md — hermes-console

> Unified UI for omostation · React 18 + TypeScript

如果你不是 Claude Code，从这里开始。

## 快速开始

```bash
bun install         # 安装依赖
bun run dev         # 开发服务器
bun run build       # TypeScript 检查 + 构建
bun test            # 运行测试
```
